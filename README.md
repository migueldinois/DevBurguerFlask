## 💻 Sobre o Projeto

O **DevBurguer** é uma landing page desenvolvida para uma hamburgueria fictícia, focada em oferecer uma experiência visual moderna e apetitosa.

Este projeto foi desenvolvido como parte do desafio de **Front-End**, com o objetivo de praticar conceitos fundamentais de CSS Moderno, Layouts Responsivos e boas práticas de organização de código.

---

## 📱 Layout e Responsividade

O projeto é totalmente responsivo, adaptando-se perfeitamente a Desktops, Tablets e Smartphones.

---

## 🛠️ Tecnologias e Conceitos Aplicados

* **HTML5 Semântico:** Uso correto das tags (`header`, `section`, `article`, `footer`) para melhor acessibilidade e SEO.
* **CSS Variables:** Criação de um arquivo `variables.css` para padronização de cores e fontes.
* **CSS Grid Layout:** Utilizado na seção de "Destaques" para criar um grid de produtos robusto (3 colunas no Desktop, 1 no Mobile).
* **CSS Flexbox:** Utilizado no Cabeçalho (`Header`), na seção de "Escolha Rápida" e no alinhamento interno dos cards.
* **Media Queries:** Breakpoints configurados para Tablets (1024px) e Mobile (768px/600px).
* **Microinterações:** Efeitos de `:hover` com `transform` e `transition` para dar vida aos botões e cards.

---

## 📂 Estrutura de Arquivos

A organização do CSS foi dividida para garantir escalabilidade:

* `reset.css`: Limpeza dos estilos padrão do navegador.
* `variables.css`: Definição da paleta de cores e tipografia.
* `main.css`: Estilos gerais e estrutura do desktop.
* `media.css`: Regras de responsividade (Mobile/Tablet).
